# Security Policy - Reporting a Vulnerability

[![banner](/.internals/trademarks/animated-banner_1200x100.svg)](#)

We take security issues **VERY SERIOUSLY AND WITHOUT CAUSING ANY PUBLIC PANICS**.
If you find any security vulnerability (both with suspicious or confirmed
target), **please STRICTLY head to this repository and report it**:

[(Holloway) Chew, Kean Ho's Security Portal](https://github.com/ChewKeanHo/Security)

> [!IMPORTANT]
>
> Security countermeasures and remedies **need time to develop**.
>
> **DO NOT BLAST IT OUT TO THE PUBLIC WITHOUT ANY REMEDY**.
>
> Please **BE PROFESSIONAL AND CIVIL especially when proper channel is
> clearly defined and available.**

Thank you.
